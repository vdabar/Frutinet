# Frutinet
Frutinet is .Net core application based on DDD and CQRS built in Microservices architecture. Commands are handled by message queu (RabbitMq).

# Technology
C#
RabbitMq
ASP.NET Core
MongoDB
FluentValidation
DDD
CQRS

# Version supported:
.NET Core 2.0
2.0 SDK Version
Visual Studio 2017.3+ (.csproj)
